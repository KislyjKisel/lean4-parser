# Parser Combinators

A parser combinator library for [Lean 4](https://leanprover.github.io/).

## Installation

Add this dependency to your project's `lakefile.lean`:

```lean
require Parser from git "https://github.com/fgdorais/lean4-parser.git" @ "main"
```
